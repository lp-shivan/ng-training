Limetools - Angular ERP

Limetools is a simple ERP build with Angular 2/4 using component based architecture

Features

  • HR Module - Managing Employees/Departments
  • Finance Module - Managing Employees Salaries
  • Admin Module - Managing users, roles and permissions
